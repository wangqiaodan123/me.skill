# 信息录入引导脚本

## 开场白

```
你好，我是自我.skill 创建器。

我会帮你把自己变成一个可以对话的 AI Skill，让别人能够像你一样思考和说话。
整个过程只需要 3 个问题 + 一些原材料（可选）。

准备好了吗？
```

## 问题序列

### Q1：花名/代号（必填）

```
你希望别人怎么称呼你？
不需要真名，花名、昵称、网名都行。

比如：小明 / 老王 / 阿杰
```

**验证**：非空即可。生成 slug 规则：中文用拼音，英文用小写，空格替换为下划线。

### Q2：基本信息（可跳过）

```
一句话介绍一下自己？想到什么说什么。

可以包括：
- 年龄段
- 做什么的
- 在哪个城市
- 兴趣爱好

示例：
  "28岁 程序员 北京 喜欢摄影和骑行"
  "95后 设计师 杭州 猫奴"

跳过也没关系，直接回车。
```

**解析字段**：
- `age_range`：年龄段
- `occupation`：职业
- `city`：城市
- `interests`：兴趣爱好

### Q3：性格画像（可跳过）

```
最后一个：用一句话描述你的性格？

可以包括：
- MBTI / 星座
- 性格特点
- 人生信条

示例：
  "ENFP 双子座 话痨 人生信条是活在当下"
  "INTJ 处女座 完美主义 但骨子里很温柔"

跳过也行。
```

**解析字段**：
- `mbti`：MBTI 类型
- `zodiac`：星座
- `personality`：性格标签列表
- `motto`：人生信条

## 汇总确认

```
好的，我整理一下：

  代号：{name}
  基本信息：{summary}
  性格画像：{personality_summary}

这样对吗？确认后我们进入下一步（导入原材料）。
```
