# 人生记忆分析器

## 任务

从原材料中提取你的人生记忆和个人特征，构建 Life Memory 知识库。

## 提取维度

### 1. 人生时间线
- 求学经历
- 职业变化
- 重要转折点
- 里程碑事件
- 当前阶段

### 2. 日常模式
- 聊天活跃时段
- 社交频率
- 生活习惯
- 作息规律

### 3. 重要经历
- 去过的地方
- 做过的事
- 旅行记忆
- 人生大事
- 难忘的经历

### 4. 饮食偏好
- 爱吃什么 / 不吃什么
- 常去的餐厅
- 做饭习惯
- 美食态度

### 5. 兴趣爱好
- 音乐/电影/书/游戏
- 日常爱好
- 技能特长
- 会主动分享什么

### 6. 人际互动模式
- 跟朋友怎么相处
- 社交风格（社牛/社恐）
- 冲突处理方式
- 在不同社交场景的表现

### 7. 价值观与信念
- 人生观
- 对感情/事业/友情的看法
- 底线和原则
- 对待世界的态度

### 8. 个人特征
- 口头禅
- 标志性行为
- 独特习惯
- 个人标签

## 输出格式

```markdown
## 人生时间线
- {时间} {事件}

## 日常模式
{描述}

## 重要经历
{按时间线列出关键经历}

## Inside Jokes / 个人梗
{只有你和朋友之间懂的梗、暗号、代称}

## 饮食偏好
{你的饮食画像}

## 人际互动模式
{描述社交风格和互动方式}

## 价值观与信念
{核心信念和态度}

## 个人特征
{口头禅、标志性行为、独特习惯}
```

## 注意事项

- 聊天记录中的事实优先于用户口述（口述可能被美化或淡化）
- 注意提取"反复出现"的模式，而不是一次性事件
- 时间信息尽量精确（从聊天记录时间戳推断）
